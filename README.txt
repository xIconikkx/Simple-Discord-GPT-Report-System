How to create a webhook?

Step 1: Create your on Discord Server
Step 2: Create a text channel
Step 3: Open Server Settings For your Discord Server
Step 4: Go To Intergrations
Step 5: Create a Webhook
Step 6: Name your Webhook, Change Icon (Not Needed)
Step 7: Select the CHANNEL you want your REPORTS to go to
Step 8: COPY WEBHOOK URL
Step 9: PASTE Webhook URl into Server.Lua 
Step 10: Test :)

Thanks, Enjoy.
